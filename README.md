# Lumina WiZ Controller 💡

Lumina WiZ Controller is a modern, minimalist desktop application designed to control your WiZ smart lights over your local network (UDP). It is built in Python using the `customtkinter` library to provide a sleek, dark-themed user interface.

## ✨ Key Features
* **Full Control:** Adjust light state (ON/OFF), brightness, and color temperature (2200K - 6500K).
* **RGB Color Picker:** Change the color of your lights using a native color picker and keep a history of your most recently used colors.
* **Auto-Discovery:** Automatically scans and finds WiZ lights on your local Wi-Fi network.
* **Integrated Renaming:** Change the name of your home or individual lights without annoying external pop-ups (features an in-app modal overlay).
* ** Responsive Design:** The application window automatically adjusts its height based on the number of discovered lights.
* **Bilingual Support:** Switch between English and Spanish in real-time.

## 🛠️ Prerequisites
* Windows OS
* Wiz E27 Lamp (Firmware 1.37.0 Tested)
